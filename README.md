# Roundcube Mailcow F2B Plugin

Add into directory called `mailcow_f2b` in roundcube plugin directory and set at least the Redis password variable in roundcube config as described in the comment of the PHP file and enable the plugin in the same config file.
Then add `roundcube: failed login from ([a-fA-F0-9:\.]+)` to Mailcow Fail2Ban Regex list.
